NBD Biblion

Goal: Clearer UI for renting books at a High school
Tasks: UI/UX design, prototype, Visual Interface Design.

A book directory application that should accompany high school students by looking for a book to rent. Task from NBD Biblion to create an interface with more visual elements to create a more playful ambience for the students.

Problem
Users who are returning to a destination they visited in the past who wants to discover new places but also returning to visited places.


Solution
Reseek helps returning travellers to indicate which places the user has visited and which places the user wants to visit. New locations are determined with suggestions related to places visited or by filtering to those needs.
